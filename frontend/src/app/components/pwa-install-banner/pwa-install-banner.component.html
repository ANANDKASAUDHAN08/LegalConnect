Install LegalConnect App

Install LegalConnect directly to your desktop or mobile home screen:

1

Look at Address Bar Icon

Click the Install Icon at the right side of your browser's address bar (Chrome, Edge, Brave).

2

Or via Browser Menu

Click browser menu (⋮) ➔ Save & Share ➔ Install LegalConnect.

Install on iPhone / iPad

Follow these quick steps in Safari to install LegalConnect as a home screen app:

1

Tap the Share Button

Tap the Share icon ⎋ in Safari's bottom bar.

2

Select 'Add to Home Screen'

Scroll down the menu options and tap 'Add to Home Screen'.

3

Launch App Anytime

Tap 'Add' at the top right to launch LegalConnect directly from your home screen.
